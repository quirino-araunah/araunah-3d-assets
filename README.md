# Araunah 3D Assets Repository

Repositorio centralizado de modelos 3D para o catalogo de equipamentos Araunah Agua.

## Estrutura

```
glb/
  tripo3d/       # 18 modelos gerados via Tripo3D image-to-3D (foto real catalogo)
  polyhaven/     # 7 modelos CC0 (tubulacao, valvulas, hidrante, barril)
  kenney/        # 2 modelos CC0 (tanque industrial, chamine)
  github/        # 2 modelos open-source (WaterBottle GLB, bomba manual STL)

thumbnails/
  catalogo/      # 18 fotos reais dos equipamentos Araunah
  sketchfab/     # 18 thumbnails de modelos Sketchfab para referencia
```

## Sketchfab (requer conta gratis pra baixar GLB)

Thumbnails salvas em `thumbnails/sketchfab/`. Para baixar os GLBs:
1. Criar conta gratis em sketchfab.com
2. Baixar cada modelo (links no catalogo-review.html)
3. Colocar em `glb/sketchfab/`

Modelos disponiveis:
- Bomba Centrifuga (ENGIPROS, CC-BY)
- Sand Filter Tank (tanphan1105)
- RO Unit (giacar25, CC-BY)
- Dosing Pump (lexcru.manan)
- Mineral Dosing System (tanphan1105)
- RO 1000L/h, RO 8m3/h, Fertigation, 7-Core Filter (tanphan1105)
- Inside RO (ramyaasubramaniam3)
- Water Treatment Plant (novusod, CC-BY)
